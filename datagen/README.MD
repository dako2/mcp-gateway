## Datagen Pipeline (Toucan-based, modified for Grok)

Generate tool-use training data using xAI Grok + real MCP servers.

### Prerequisites

- `XAI_API_KEY` set in `../.env`
- MCP server files in `../mcp_servers/` (generated by `src/convert_to_toucan.py`)
- Gateway running for tool calls: `cd ../src && uvicorn gateway:app --port 8000`

### Pipeline Steps

#### Step 1: Question Synthesis

```bash
# 1.1 Generate question prompts (no LLM needed)
python step1.1_gen_questions.py --total_prompts 1000 --sampling_strategy random

# 1.2 Generate questions using Grok
bash step1.2_completion.sh <file_from_1.1>

# 1.3 Deduplicate and clean
python step1.3_process_completion.py --input_file <file_from_1.2>
```

#### Step 2: Question Quality Check

```bash
# 2.1 Generate quality check prompts
python step2.1_question_quality_check.py --input_file <*_3sanitized.jsonl>

# 2.2 Rate quality using Grok
bash step2.2_completion_quality_check.sh <file_from_2.1>

# 2.3 Extract ratings and filter
python step2.3_process_completion.py --input_file <file_from_2.2>
```

#### Step 3: Agent Trace Generation

```bash
# 3.1 Generate agent trajectories (calls real MCP tools via gateway)
bash step3.1_completion_agent.sh <*_2prepared.jsonl>

# 3.2 Filter bad traces
python step3.2_process_completion.py --input_file <file_from_3.1>
```

#### Step 4: Response Quality Check

```bash
# 4.1 Generate response quality check prompts
python step4.1_response_quality_check.py --input_file <file_from_3.2>

# 4.2 Rate responses using Grok
bash step4.2_completion_response_check.sh <file_from_4.1>

# 4.3 Extract ratings and final filtering
python step4.3_process_completion.py --input_file <file_from_4.2>
```

#### Multi-Turn Conversations

```bash
bash ext_completion_multi_turn.sh <input_file> grok-4-1-fast-reasoning xai 5
```

### Wrapper Script

```bash
# Shortcut for any LLM step
bash run_with_grok.sh completion <input_file>    # Steps 1.2, 2.2, 4.2
bash run_with_grok.sh trace_gen <input_file>     # Step 3.1
```
